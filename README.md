# My_ide

## Project Goals

Develop a cross platform ide capable of running on mac, windows, linux, and the browser.


## Features

- Create, open, and edit text files
- Display text
- Syntax highlighting
- Theme editing
- Compile and run programs
- Support to multiple languages
- Environment management
- Debugging tools
	- Break points
	- Stack Explorer
	- Symbols Explorer
	- Error highlighter
- Code management tools
	- Symbol editing
	- File Explorer
- Multi-plataform
- Plugin system
- Git integration
